1. board_json : 게시판의 글들을 json 타입을 보내주기 위해 쓰인 php.

2. check : 데이터베이스에 존재하는 계정인지 확인하고 로그인으로 되었다는 것을 체크해주기 위한 php.

3. dbcon : 데이터베이스에 연결해주기 위한 php.

4. delboard : 게시판에 등록된 글을 삭제하기 위해 쓰인 php.

5. findid : 이메일 값을 통해 아이디 값을 찾기위함.

6. findpw : 비밀번호를 찾을 때, id와 이메일 값을 통해 비밀번호를 찾기위함

7. img : php에 등록된 사진을 안드로이드에 불러오기 위해 쓰인 php.

8. insert : 회원가입을 위해 쓰인 php.

9. json_img : 선택된 이미지를 json 타입으로 보내주기 위해 쓰인 php.

10. Login : 로그인시, 계정확인을 위함.

11. setboard : 게시판에 등록된 글을 뽑아내기 위해 쓰인 php.

12. test : 이미지를 넣기 위해 쓰인 php.

13. upload : 사진을 업로드 하기 위해 쓰인 php (품목명 2개를 입력하여 각각 한 url의 쿼리문안에 삽입하도록 도와줌)

14. withdraw : 탈퇴하기 위한 php.

15. write_board : 글을 작성하기 위한 php.
